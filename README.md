# Smart Finance Calculator

A modern web application built using React and Vite.

## Project Overview

Smart Finance Calculator helps users understand their monthly financial position by calculating income, expenses, savings, and savings percentage.

## Calculator Branch

This branch contains the finance calculator project setup and calculation functionality.

## Features

- Monthly income calculation
- Monthly expense calculation
- Savings calculation
- Savings percentage
- Financial summary
- Smart finance tips
- Responsive dashboard design
